# Get the Gene Ontology to gene connections

Get the Gene Ontology to gene connections

## Usage

``` r
get_gene_to_go(head_only = FALSE)
```

## Arguments

- head_only:

  Logical. If TRUE, returns only the first 5 rows.

## Value

data.table with the Gene Ontology to gene edges

## Details

The data is extracted from the OpenTargets release `25.12`
