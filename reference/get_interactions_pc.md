# Get the pathway commons network

Get the pathway commons network

## Usage

``` r
get_interactions_pc(head_only = FALSE)
```

## Arguments

- head_only:

  Logical. If TRUE, returns only the first 5 rows.

## Value

data.table with the Pathway Commons network data.

## Details

This is version 14 of Pathway Commons. For details, check Rodchenkov, et
al., Nucleic Acids Res., 2019
