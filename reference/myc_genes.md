# MYC target genes

The MYC target genes from the MSigDB (Hallmarks V1).

## Usage

``` r
myc_genes
```

## Format

### `myc_genes`

A data.table with 200 rows and 3 columns:

- ensembl_gene:

  Ensembl identifier

- gene_symbol:

  Gene Symbol

- gene_set:

  Name of the gene set
