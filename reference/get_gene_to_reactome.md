# Get the Reactome to gene connections

Get the Reactome to gene connections

## Usage

``` r
get_gene_to_reactome(head_only = FALSE)
```

## Arguments

- head_only:

  Logical. If TRUE, returns only the first 5 rows.

## Value

data.table with the Reactome to gene edges

## Details

The data is extracted from the OpenTargets release `25.12`
