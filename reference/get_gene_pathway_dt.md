# Get the gene to pathway data.table

Get the gene to pathway data.table

## Usage

``` r
get_gene_pathway_dt(object)
```

## Arguments

- object:

  The `GeneWalk` class, please see
  [`GeneWalk()`](https://gregorlueg.github.io/genewalkR/reference/GeneWalk.md).

## Value

Returns the gene to pathway data.table stored in the object
