# Get the Gene Ontology main information

Get the Gene Ontology main information

## Usage

``` r
get_gene_ontology_info(head_only = FALSE)
```

## Arguments

- head_only:

  Logical. If TRUE, returns only the first 5 rows.

## Value

data.table with the Gene Ontology information

## Details

The data is extracted from the obo file downloaded from [GO OBO
Foundry](https://obofoundry.org/ontology/go.html). Last download:
`January '26`.
